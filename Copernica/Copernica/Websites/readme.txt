De naam van de pagina.html is de template pagina.
De stylesheet file is de stylesheet voor die template(s).
De tekst input.html file is de content die op de pagina geplaatst staat.

De stylesheet van mailvoorkeuren wordt voor meerdere pagina's gebruikt, namelijk:
  - Bedankt
  - Feedback
  - Klant extra's
  - Mailvoorkeuren
  - Profielverrijking
  - Stuurdoor
  - Tellafriend
  - testpagina
  - Uitschrijven
